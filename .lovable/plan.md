
## Página Bio — "Cansei de Ser Gordo"

Landing page responsiva e moderna para Jorge Bentes com animações de entrada e estilo vibrante.

### Design
- **Cores**: Verde limão (#CDFF00) + Laranja vibrante (#FF6600) como cores principais, fundo escuro para contraste
- **Tipografia**: Bold e impactante, estilo energético
- **Animações**: Fade-in e slide-up nos elementos ao carregar, hover effects nos botões, pulse no logo

### Estrutura da página

1. **Hero / Topo**
   - Logo/imagem do "Cansei de Ser Gordo" (usar a imagem circular enviada)
   - Nome "Jorge Bentes" e tagline do projeto
   - Animação de entrada suave

2. **Seção Sobre/Bio**
   - Breve texto sobre o Jorge e a missão do projeto (placeholder editável)
   - Ícone de infinito da marca

3. **Botões de Links**
   - 🎬 YouTube → link do canal
   - 📸 Instagram → link do perfil
   - 📱 WhatsApp → link wa.me (placeholder para número)
   - Botões grandes, arredondados, com ícones e efeitos hover animados

4. **Footer**
   - Créditos simples

### Assets
- Copiar as imagens enviadas para o projeto (logo circular e banner)
- Usar as imagens reais na página

### Responsividade
- Mobile-first, layout centralizado em coluna
- Adapta bem em desktop com largura máxima contida
